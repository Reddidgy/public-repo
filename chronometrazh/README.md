# Chronometrazh - Text Timing Calculator

A simple, client-side web application to estimate the speaking or reading time for a piece of text. This tool is useful for speakers, content creators, and anyone who needs to quickly time a script.

The calculation is performed in real-time directly in your browser as you type.

## Features

- **Real-Time Estimation**: Calculates the estimated time (in minutes and seconds) as you type.
- **Word Count**: Displays the total word count of your text.
- **Sleek UI**: A clean, modern interface with a dark theme that's easy on the eyes.
- **Client-Side**: Runs entirely in the browser. No data is sent to a server, ensuring your privacy.
- **No Installation**: Pure HTML, CSS, and JavaScript. Just open the file and use it.

## How to Use

1.  Save the code as an `index.html` file.
2.  Open the `index.html` file in your favorite web browser.
3.  Begin typing or paste your text into the text area.
4.  The estimated speaking time and word count will update automatically below the input field.
